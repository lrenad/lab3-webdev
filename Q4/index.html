Quantum
Bit
Commitment

Abstract

Quantum bit commitment is deemed impossible due to the use of entanglement by a dishonest participant. We show in this paper how the use of entanglement can be detected by the other party, therefore making secure quantum bit commitment possible. To this end, we rely on the fundamental property of an EPR pair to create correlations when measured, therefore introducing order into what should otherwise be a randomly generated binary string. This result opens the door for a whole set of cryptographic applications using bit commitment as a building block: remote coin tossing, zero-knowledge proofs and secure two-party computation.
